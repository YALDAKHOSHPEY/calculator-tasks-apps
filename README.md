# 🧮 Calculator & 📝 Task Apps

A simple and responsive collection of JavaScript mini-apps — including a **Calculator** and a **To-Do List App** — designed for hands-on practice and clean UI experience.

## 🚀 Live Demo

> Coming soon... (You can deploy it with GitHub Pages or Netlify!)

---

## ✨ Features

### 🔢 Calculator App
- Basic arithmetic operations
- Clear & delete functions
- Responsive design
- Clean and intuitive interface

### ✅ To-Do List App
- Add, edit, delete tasks
- Mark tasks as completed
- Filter tasks by status
- Local storage support
